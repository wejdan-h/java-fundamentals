# School Management System (OOP Project)

## Overview
This project is an Object-Oriented Programming system that models a school structure including employees and students with different levels.

## Classes
- School
- Employee
- Student
- Teacher
- Administrator
- PrimaryStudent
- IntermediateStudent
- HighSchoolStudent

## Concepts Used
- Inheritance
- Encapsulation
- Method overriding (if applied)
